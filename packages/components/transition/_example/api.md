
### Transition Props
名称 | 类型 | 默认值 | 说明 | 必传
-- | -- | -- | -- | --
appear | Boolean | false | 首次出现是否展示动画 | N
customClass | String | '' | 自定义容器类名 | N
destoryOnClose | Boolean | false | 隐藏时是否销毁内容 | N
duration | Number | - | 指定过渡时间 | N
name | String | 't-transition' | 过渡类名 | N
visible | Boolean | false | 是否显示 | N
